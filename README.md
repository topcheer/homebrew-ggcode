# homebrew-ggcode

Homebrew tap for [ggcode](https://github.com/topcheer/ggcode) — an AI coding agent for the terminal.

## Install

```bash
brew tap topcheer/ggcode
brew install ggcode
```

## Upgrade

```bash
brew upgrade ggcode
```

## Uninstall

```bash
brew uninstall ggcode
brew untap topcheer/ggcode
```
